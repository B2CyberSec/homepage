# Light Theme Palette (von Referenz b2cybersec-jbtggu4e.manus.space)

## Apple-Inspired Light Design
- **Background (primary):** #ffffff
- **Background secondary / cards subtle:** #fbfbfd
- **Surface / muted / sidebar:** #f5f5f7
- **Surface alt:** #f8f8f7
- **Foreground (primary text):** #1d1d1f
- **Foreground secondary:** #34322d
- **Muted text:** #86868b / #858481
- **Border:** rgba(0,0,0,0.08) (#00000014)
- **Border strong:** #d2d2d7

## Akzentfarben
- **Primary (Blau):** #2563eb (B2 Blue Light)
- **Primary hover:** #1d4ed8
- **Primary subtle bg:** #2563eb1a (10% alpha)
- **Primary border:** #2563eb33
- **Orange Accent (CTA secondary):** #e8530e / hover #c44400
- **Destructive / red:** #dc2626 / #ff3b30
- **Dark contrast bg:** #1a1a19 / #1d1d1f (für inverted CTA Sektionen)

## Typografie
- **Font Family:** Inter (preconnect zu fonts.googleapis.com)
- weight 300-900

## Stil-Charakteristika
- Sehr großzügiges Weißraum
- Massive Typografie
- Schatten dezent (shadow-2xl/black 5%)
- Karten: bg-white mit dezenter border + sehr weichem Schatten
- Glas-Cards: bg-white/80 oder /90 mit backdrop-blur
- Nav: white/97 mit blur und subtiler border

# B2CyberSec Website

Die offizielle Website der **B2CyberSec GmbH** — IT-Security so einfach wie bei Amazon.

## Tech Stack

- **React 19** + TypeScript
- **Vite** (Build Tool)
- **Tailwind CSS 4** + shadcn/ui
- **Wouter** (Client-Side Routing)
- **Framer Motion** (Animationen)
- **Lucide React** (Icons)

## Seiten

| Seite | Route | Beschreibung |
|-------|-------|-------------|
| Startseite | `/` | Hero, NIS-2 Urgency, Featured Services, How It Works, Trust |
| Services | `/services` | Vollständiger Service-Katalog mit Preisen und Bewertungen |
| NIS-2 | `/nis-2` | Dedizierte NIS-2 Compliance Landing Page |
| Pentesting | `/pentesting` | Penetrationstest Services |
| MSSP | `/mssp` | Managed Security Services |
| Experten auf Zeit | `/experten` | Staff Augmentation (Nearshore/Onshore) |
| Über uns | `/ueber-uns` | Team und Unternehmen |
| Kontakt | `/kontakt` | Calendly-Integration, Kontaktdaten |
| Impressum | `/impressum` | Rechtliche Angaben |
| Datenschutz | `/datenschutz` | Datenschutzerklärung |

## Entwicklung

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
